值得关注

  • Cursor团队版改为20美元基础+按量付费，引发成本担忧
  • Claude Code Max被推荐为高性价比替代方案，200美元/月可支撑百人团队
  • 用户反馈Cursor响应变慢，疑因路由架构瓶颈

潜在机会

  • 过度依赖按量付费工具可能导致预算失控
  • 使用非官方渠道（如crs）存在封号风险
